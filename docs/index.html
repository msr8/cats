This is a simple site for 7k+ images and videos scraped from 70+ cat related subreddits. Press the "Random Cat" button below to view one of said images/videos. You can constantly refresh that page (ctrl+R or cmd+R) to load new cat images/videos. This website is open source and the frontend is made using HTML+CSS+JS, while the data is scraped using python
